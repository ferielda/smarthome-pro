# SmartHome Pro 智能家居控制系统

这是一个基于React 19 + TypeScript开发的智能家居控制 APP，旨在提供简洁、高效的家庭设备管理体验。

## 🚀 主要功能
* **设备远程控制**：支持灯光、空调、窗帘等智能设备的一键开关。
* **场景模式**：一键切换“回家模式”、“离家模式”、“影院模式”。
* **实时状态监测**：通过 WebSocket 实现家庭传感器数据的实时更新。

## 🛠 如何安装与运行

确保你的电脑已安装 [Node.js](https://nodejs.org/)，然后在终端执行：

1. **克隆代码到本地**
   git clone [https://github.com/ferielda/smarthome-pro.git](https://github.com/ferielda/smarthome-pro.git)
2. 进入项目目录
   cd smarthome-pro
3.安装依赖
   npm install
4.启动开发服务器
   npm run dev
    
应用通常会在 http://localhost:5173 (或 3000) 自动打开。
