# SmartHome Pro 智能家居控制系统

这是一个全栈智能家居控制应用，结合了现代化的前端交互体验与后端数据管理能力。

## 🚀 项目技术栈

* **前端框架**: React 19 + TypeScript
* **构建工具**: Vite 6
* **样式库**: Tailwind CSS 4
* **后端服务**: Express.js
* **数据库**: Better-SQLite3
* **AI 能力**: Google Gemini API (@google/genai)
* **动画库**: Motion (framer-motion)

## 🛠 功能亮点

* **智能家居面板**: 实时控制家庭设备状态。
* **AI 智能助手**: 集成 Google Gemini AI，支持自然语言解析家居指令。
* **数据持久化**: 使用 SQLite 数据库存储设备状态与配置。
* **现代化 UI**: 基于 Tailwind CSS 与 Lucide 图标库的响应式设计。

## 📦 如何安装与运行

### 1. 克隆代码
```bash
git clone [https://github.com/ferielda/smarthome-pro.git](https://github.com/ferielda/smarthome-pro.git)
cd smarthome-pro

```

### 2. 安装依赖

确保你的电脑已安装 [Node.js (v18+)](https://nodejs.org/)：

```bash
npm install

```

### 3. 配置环境变量

在项目根目录下创建一个 `.env` 文件，根据需要配置你的 API 密钥：

```text
# 例如:
GEMINI_API_KEY=your_api_key_here

```

### 4. 启动项目

```bash
npm run dev

```

项目启动后，请访问终端中显示的本地地址（通常是 `http://localhost:3000`）。

## 📝 开发与维护

* **构建项目**: `npm run build`
* **类型检查**: `npm run lint`
* **数据库**: 项目使用 `better-sqlite3`，数据库文件将自动创建在项目目录中。
